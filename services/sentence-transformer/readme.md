# sentence-transformer

ML-модель для вычисления релевантности текстов.

## REST

### Health check

`GET /health`

Алгоритм работы:

1. Всегда возвращает `HTTP 200` с объектом `{"status":"READY"}`
2. При любом исключении возвращает `HTTP 503` с текстом исключения в теле ответа

### Вычислить векторное представление текста

`POST /text/vectorize`

|Входной параметр|Источник|Описание|
|----|----|----|
|📌 `text`|атрибут тела запроса|Корпус текста, векторное представление которого требуется вычислить|

Алгоритм работы:

1. Запрашивает у ML-модели длину входного текста в токенах и длину контекстного окна модели
   1. Если длина текста превышает длину контекстного окна, возвращает `HTTP 413`
2. Передает ML-модели текст, получает вектор (массив вещественных чисел)
   1. Возвращает вектор с `HTTP 200`
3. В случае любого исключения возвращает `HTTP 500` с текстом исключения в теле

### Вычислить косинусное сходство двух векторов

`POST /vectors/cosine-similarity`

|Входной параметр|Источник|Описание|
|----|----|----|
|📌 `left`|атрибут тела запроса|Массив вещественных чисел|
|📌 `right`|атрибут тела запроса|Массив вещественных чисел|

Алгоритм работы:

1. Вычисляет косинусное сходства `left` и `right`
2. Возвращает вещественное число от 0 до 1 с `HTTP 200`
3. При любом исключении возвращает `HTTP 500` с текстом исключения в теле
